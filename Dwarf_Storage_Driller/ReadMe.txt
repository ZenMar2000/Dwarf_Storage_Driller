How to use:
1) Select a path using the button or writing it in the text field

2) Click on 'New Drill' button to begin

3) To drill further, select one or more rows using the very first column and click the 'Drill Down' button

4) If you want to focus on some folders, select one or more rows and press the 'Scope rows' button
	It will filter all children rows inside the scoped rows

5) To remove the filter, click on 'Remove scope' button
